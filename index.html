STGTour.GTour

Welcome to STGTour.GTour Blazor Guided UI Tour Service. This package was inspired by all the Angular npm packages that are so freely available for quick starts and not so much for Blazor.

Purpose

The STGTour GTour package is a customizable UI Tour Component intented to help onboarding users within your Blazor application by giving them a chance to interact with your system while showing them important tips and how to's. A tour can be started at any time and it's up to the developer to decide when and how it's started. It can be used to show specific sequences required by an application or just a general feel of where to find controls and general functionality
